Finding Ghost Particles

Applying machine learning to waveform data from the Majorana Demonstrator to improve event identification and energy estimation.

Get started View project repo Our report
This site summarizes our DSC 180 capstone project. Regression is in progress and will be added as results are finalized.

Overview

We analyze detector waveforms from the Majorana Demonstrator and train machine learning models to distinguish signal-like single-site events from background-like multi-site events. We also explore regression for predicting continuous targets such as energy.

Data

Input: time-series detector waveforms. Each waveform is represented as a fixed-length sequence of samples.

  • Waveforms stored in HDF5 files
  • Derived tabular features saved to CSV

Key challenges:

  • Class imbalance for several labels
  • Noise and baseline drift across detectors
  • Feature stability across files

Features

We extract waveform features capturing pulse shape, tail behavior, and frequency content.

  • LQ80, ND80
  • Half-Width Peak, PPR, Energy Duration
  • Spectral Centroid and other FFT-based features

Methods

Classification

We trained baseline and tree-based classifiers and evaluated using F1, precision, recall, and confusion matrices. We also performed threshold tuning when needed.

  • Baseline: Logistic Regression
  • Improved: Random Forest (and comparisons)

Regression

Regression is in progress. We will establish a baseline regressor and compare against improved models.

  • Baseline: Linear / Ridge Regression
  • Improved: Random Forest Regressor / Gradient Boosting

Results

Classification results will be summarized here with tables and plots (confusion matrix, feature importance). Regression results will be added once finalized.

  • Final model: Random Forest for psd_label_lq (example)
  • Main metric: F1 score due to imbalance
